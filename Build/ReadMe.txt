ScreenShot폴더 : 프로그램 실행중 PrintScreen키를 눌렀을때의 화면이 이미지로 저장되는 곳

Data폴더

  |- LoadData폴더 : 로드할 리소스들의 경로가 있음
     |- Img폴더 : 텍스쳐경로
     |- Sound폴더 : 사운드경로
     |- Mesh폴더 : 메쉬경로
      - SknMesh폴더 : Skinned메쉬 경로

  |- WhatBox폴더 : 프레임워크에서 필요로하는 정보나 리소스들이 있음
     |- Img폴더 : 프레임워크에서 쓰는 텍스쳐들
      - LoadingImg폴더 : 로드화면에서 쓰는 텍스쳐들

Util폴더

  |- BinaryEditor.exe : FileStream 형식대로 파일을 이진모드로 읽거나 쓰는 프로그램
  |- DotEditor.exe : cShp_Polygon과 호환되는 형식의 점 배열데이터 파일을 만들어 주는 프로그램
